## 8. 不要重复踩的坑

- `dotenv` 默认不会覆盖已有 shell 环境变量，当前代码已改成 `override: true`
- Feishu Node SDK 的 `im.message.receive_v1` 是扁平结构，不是 `data.event.message`
- 群聊默认要求 `@机器人`，但当前代码已对“未配置 botOpenId/botName”做了兼容
- `/bind` 现在支持绝对路径，但之前不支持，不要再退回去
